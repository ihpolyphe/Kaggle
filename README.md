# Kaggle

## About
Kaggleで作成したコードを管理するリポジトリ

## Environment
実行環境：Annaconda

## Setup
make conda env
```
conda create -n kaggle_env python=3.8
```
activate
```
conda activate kaggle_env
```

install conda library reference [conda_install.md](/conda_install.md)

install pip library
```
pip install -r install_requiremnets.txt
```
